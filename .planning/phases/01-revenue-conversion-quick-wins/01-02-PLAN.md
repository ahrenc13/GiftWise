Static product dict shape (all fields required):
```python
{
    "title": str,          # Product display name
    "snippet": str,        # 1-2 sentence description shown in recommendations
    "price": str,          # Format: "$NNN.NN"
    "source_domain": str,  # Bare domain, e.g. "youngelectricbikes.com"
    "link": str,           # Awin deep link (see format below)
    "image_url": str,      # Empty string "" — images are resolved programmatically
    "interest_match": str, # Space-separated interest keywords for scoring
}
```

Awin deep link format:
```
https://www.awin1.com/cread.php?awinmid=ADVERTISER_ID&awinaffid=!!id!!&agen=&clickref=&p=ENCODED_PRODUCT_URL
```
- `!!id!!` is the publisher ID placeholder (existing convention — do NOT replace with a real ID)
- `ENCODED_PRODUCT_URL` is the full product page URL, percent-encoded
- Example from OUTFITR: `https://www.awin1.com/cread.php?awinmid=117613&awinaffid=!!id!!&agen=&clickref=&p=https%3A%2F%2Foutfitrer.com%2Fproducts%2Ffoldable-2-bike-hitch-rack-platform-style-2-receiver-200-lbs-capacity`

_get_awin_static_products() injection pattern (one block per merchant):
```python
# MERCHANT_NAME — trigger on THEME themes
merchant_triggers = {"keyword1", "keyword2", ...}
if any(t in interest_text for t in merchant_triggers):
    results.extend(_MERCHANT_ALL_PRODUCTS)
```
